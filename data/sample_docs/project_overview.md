# VeriDocs Sample Document

VeriDocs is a classroom AI demo that combines retrieval and generation.

## Goals
- Show a simple RAG architecture
- Use an open-weights Hugging Face model
- Return citations with each answer
- Add a verification status and confidence score

## Core Model
The generation model used in this project is `google/flan-t5-small`.

## User Flow
1. Ingest documents
2. Ask a question
3. Review answer, citations, and evidence quotes
4. Check verification status
